# NeovimConfig

## Introduction

This is my personal NeoVim configuration for Linux.

## Install
1. Clone this repository.
    ```shell
    git clone https://github.com/RohanSiddhu/NeovimConfig.git
    ```
2. Now, cd to 'NeovimConfig'.
    ```shell
    cd NeovimConfig
    ```
3. Run 'setup.sh'.
    ```shell
    ./setup.sh
    ```
